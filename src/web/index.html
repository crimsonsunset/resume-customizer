🚀 Resume Optimizer

🔮 Web App Coming Soon

This will be a recruiter-facing web application where hiring managers can input job descriptions and generate personalized resumes.

✨ Planned Features

  • 🤖 AI-powered job description matching
  • 📄 Interactive resume generation
  • 🎨 Multiple professional templates
  • 📊 Skills highlighting and customization
  • 📱 Mobile-responsive design
  • 🔄 Real-time PDF preview

🛠️ Current CLI Tool

# Available now - HTML to PDF conversion
npm run html-to-pdf input.html --css styles.css -o output.pdf

# Future web app development
npm run dev:web # Start Vite development server
npm run build:web # Build for production

🎯 Technology Stack

  • ⚡ Vite for modern web development
  • 🎨 Vanilla JavaScript with ES modules
  • 🔧 Path aliases for clean imports
  • 🌐 Netlify Functions for serverless backend
  • 🎭 Playwright for PDF generation
  • 📱 Modern CSS Grid layouts
